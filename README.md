# 🥒 Waapickle – Online Pickle Store

## 📌 Live Demo
🔗 https://waapickle.vercel.app/

---

## 📌 Overview
Waapickle is an online platform for selling homemade and traditional pickles. It allows users to explore different varieties, place orders, and enjoy authentic flavors delivered to their doorstep.

---

## 🚀 Features
- 👤 User registration and login  
- 🥭 Browse different types of pickles  
- 🔍 Search and filter products  
- 🛒 Add to cart and checkout  
- 📦 Order placement and tracking  
- 🛠️ Admin panel for product management  

---

## 🧠 Technologies Used
- Frontend: HTML, CSS, JavaScript  
- Backend: Python (Flask) / Node.js  
- Database: MySQL / MongoDB  

---

## ⚙️ Installation

1. Clone the repository:
```bash
git clone https://github.com/your-username/Waapickel.git
````

2. Navigate to the folder:

```bash
cd Waapickel
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

or

```bash
npm install
```

---

## ▶️ Usage

Run the application:

For Python:

```bash
python app.py
```

For Node.js:

```bash
npm start
```

Open in browser:

```
http://127.0.0.1:5000
```

or

```
http://localhost:3000
```

---

## 📊 How It Works

1. User registers/logs in
2. Browses pickle products
3. Adds items to cart
4. Places order
5. Admin manages products and orders
